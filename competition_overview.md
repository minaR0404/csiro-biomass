# CSIRO - Image2Biomass Prediction コンペティション概要

## 基本情報

| 項目 | 内容 |
|------|------|
| **主催** | CSIRO、Meat & Livestock Australia (MLA)、Google Australia |
| **プラットフォーム** | Kaggle |
| **URL** | https://www.kaggle.com/competitions/csiro-biomass |
| **形式** | Code Competition（Kernel Only） |
| **メダル** | あり |

## 目的

**牧草地の俯瞰画像からバイオマス（牧草量）を予測するAIモデルの構築**

放牧管理において、家畜が食べられる草の量を正確に推定することは、生産性・環境持続可能性・生物多様性に影響する重要な要素。現在は手作業でのサンプリングが必要だが、AIモデルによりこれを効率化することを目指す。

## データセット

### ファイル構成

```
csiro-biomass/
├── train.csv          # トレーニングデータのメタ情報とターゲット値
├── test.csv           # テストデータのメタ情報（予測対象）
├── sample_submission.csv  # 提出フォーマットのサンプル
├── train/             # トレーニング画像ディレクトリ（JPEG）
└── test/              # テスト画像ディレクトリ（スコアリング時に利用可能）
```

### train.csv カラム詳細

| カラム名 | 説明 |
|----------|------|
| `sample_id` | 各トレーニングサンプル（画像）の一意識別子 |
| `image_path` | トレーニング画像への相対パス（例: `images/ID1098771283.jpg`） |
| `Sampling_Date` | サンプル収集日 |
| `State` | サンプル収集したオーストラリアの州 |
| `Species` | 存在する牧草種（バイオマス順、アンダースコア区切り） |
| `Pre_GSHH_NDVI` | 正規化植生指数（GreenSeeker）の読み取り値 |
| `Height_Ave_cm` | 落下板で測定した平均牧草高さ（cm） |
| `target_name` | このレコードのバイオマス成分名 |
| `target` | `target_name`に対応する正解バイオマス値（グラム） |

### test.csv カラム詳細

| カラム名 | 説明 |
|----------|------|
| `sample_id` | 各予測行の一意識別子（画像×ターゲットペアごとに1行） |
| `image_path` | テスト画像への相対パス（例: `test/ID1001187975.jpg`） |
| `target_name` | 予測すべきバイオマス成分名 |

### sample_submission.csv カラム詳細

| カラム名 | 説明 |
|----------|------|
| `sample_id` | `test.csv`からコピー（画像×`target_name`ペアごとに1行） |
| `target` | 予測したバイオマス値（グラム） |

### 画像データ仕様

- **画像数**: 1,162枚（トレーニング用）、800枚以上（テスト用）
- **撮影範囲**: 70cm × 30cm のクアドラート（方形区）
- **解像度**: 2,000 × 1,000 ピクセルに統一
- **フォーマット**: JPEG
- **収集地域**: オーストラリア4州19地点
- **収集期間**: 2014〜2017年（3年間、全季節カバー）

### 予測対象（5つのターゲット変数）

| 変数名 | 説明 |
|--------|------|
| `Dry_Green_g` | クローバー以外の緑色植生の乾燥重量（グラム） |
| `Dry_Dead_g` | 枯死物質の乾燥重量（グラム） |
| `Dry_Clover_g` | クローバー成分の乾燥重量（グラム） |
| `GDM_g` | 緑色乾物 - Green Dry Matter（グラム） |
| `Dry_Total_g` | 総バイオマス（グラム）- **最重要** |

### 提出形式

`test.csv`の各`sample_id`に対して、`sample_submission.csv`に単一の数値`target`を出力。各行は1つの`(image_path, target_name)`ペアに対応し、その成分の予測バイオマス（グラム）を提供する。

## 評価指標

**グローバル加重決定係数（Weighted R²）**

- 5つのターゲット変数それぞれのR²を重要度で加重平均
- `Dry_Total_g` が最も高い重みを持つ

## 提出ルール

| ルール | 内容 |
|--------|------|
| 提出方法 | Kaggle Notebooks経由（Kernel Only） |
| 実行時間制限 | CPU/GPU合わせて最大9時間 |
| インターネット | 無効化必須 |
| 外部データ | 公開データセットのみ使用可 |
| 提出ファイル | `submission.csv`（sample_id, 予測値の2列） |

## 技術的ポイント

### データの特徴

- 複数カメラ機種（iPhone、Canon、Nikon等）で撮影
- 様々な撮影高度・照明条件
- 季節・地域・牧草種の多様性が含まれる
- 品質管理済み（初期3,187サンプルから1,162サンプルに選別）

### 考慮すべき課題

- マルチターゲット回帰問題
- 画像の多様性への対応（照明、角度、カメラ差異）
- 各ターゲット間の相関関係の活用
