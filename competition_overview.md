| 項目 | 内容 |
|------|------|
| **主催** | CSIRO、Meat & Livestock Australia (MLA)、Google Australia |
| **プラットフォーム** | Kaggle |
| **URL** | https://www.kaggle.com/competitions/csiro-biomass |
| **形式** | Code Competition（Kernel Only） |
| **メダル** | あり |

## 目的

**牧草地の俯瞰画像からバイオマス（牧草量）を予測するAIモデルの構築**

放牧管理において、家畜が食べられる草の量を正確に推定することは、生産性・環境持続可能性・生物多様性に影響する重要な要素。現在は手作業でのサンプリングが必要だが、AIモデルによりこれを効率化することを目指す。

## データセット

### 画像データ

- **画像数**: 1,162枚（トレーニング用）、800枚以上（テスト用）
- **撮影範囲**: 70cm × 30cm のクアドラート（方形区）
- **解像度**: 2,000 × 1,000 ピクセルに統一
- **収集地域**: オーストラリア4州19地点
- **収集期間**: 2014〜2017年（3年間、全季節カバー）

### 予測対象（5つのターゲット変数）

| 変数名 | 説明 |
|--------|------|
| `Dry_Green_g` | クローバー以外の緑色植生の乾燥重量 |
| `Dry_Dead_g` | 枯死物質の乾燥重量 |
| `Dry_Clover_g` | クローバー成分の乾燥重量 |
| `GDM_g` | 緑色乾物（Green Dry Matter） |
| `Dry_Total_g` | 総バイオマス（最重要） |

## 評価指標

**グローバル加重決定係数（Weighted R²）**

- 5つのターゲット変数それぞれのR²を重要度で加重平均
- `Dry_Total_g` が最も高い重みを持つ

## 技術的ポイント

### データの特徴

- 複数カメラ機種（iPhone、Canon、Nikon等）で撮影
- 様々な撮影高度・照明条件
- 季節・地域・牧草種の多様性が含まれる
- 品質管理済み（初期3,187サンプルから1,162サンプルに選別）

### 考慮すべき課題

- マルチターゲット回帰問題
- 画像の多様性への対応（照明、角度、カメラ差異）
- 各ターゲット間の相関関係の活用
