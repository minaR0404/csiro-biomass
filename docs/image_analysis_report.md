## 3. 視覚的特徴とターゲット値の関係

### 3.1 観察された相関

| 視覚的特徴 | 関連ターゲット |
|-----------|---------------|
| 緑色の面積・濃さ | Dry_Green_g, GDM_g |
| 茶色・枯れた部分 | Dry_Dead_g |
| 三つ葉の形状 | Dry_Clover_g |
| 土壌の露出度 | 総バイオマス（逆相関） |
| 植生の密度 | Dry_Total_g |

### 3.2 モデルが学習すべき特徴

1. **色ヒストグラム**: 緑/茶/土色の比率
2. **テクスチャ**: 草の密度・パターン
3. **セグメンテーション**: 緑色植生 vs 枯死物質 vs 土壌
4. **葉の形状**: クローバー（三つ葉）の検出

---

## 4. 課題と対策

### 4.1 データの課題

| 課題 | 詳細 | 対策案 |
|------|------|--------|
| 照明のばらつき | 日光・影の影響 | 色正規化、データ拡張 |
| 低バイオマス画像 | 土壌が多い画像 | 重み付けサンプリング |
| Dry_Clover_g = 0 | 多くの画像で0 | 2段階予測（有無→量） |
| Height情報なし | テストにはHeightなし | 画像からの高さ推定は困難 |

### 4.2 推奨データ拡張

- 水平・垂直フリップ
- 回転（90度単位）
- 色調・明度の調整
- ランダムクロップ（70×30cmは固定だが、解像度内でクロップ可能）

## 5. ターゲット変数の関係式検証

### 5.1 画像1（ID1011485656）

```
GDM_g = Dry_Green_g + Dry_Clover_g

Dry_Total_g = Dry_Green_g + Dry_Dead_g + Dry_Clover_g

```

## 6. モデリング戦略への示唆

### 6.1 アーキテクチャ案

```
画像 → CNN/ViT → 特徴ベクトル → マルチヘッド回帰
                                    ├─ Dry_Green_g
                                    ├─ Dry_Dead_g
                                    └─ Dry_Clover_g

後処理:
  GDM_g = Dry_Green_g + Dry_Clover_g
  Dry_Total_g = Dry_Green_g + Dry_Dead_g + Dry_Clover_g
```

### 6.2 代替アプローチ

1. **5ターゲット独立予測** → 後処理で整合性調整
2. **3ターゲット予測**（Green, Dead, Clover）→ 残り2つは計算
3. **セマンティックセグメンテーション** → ピクセル単位で分類 → 面積から重量推定
